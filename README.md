# OrangeHRM-Automation-Framework
This project is a Hybrid Automation Framework developed for testing the OrangeHRM web application. The framework is built using Selenium WebDriver, Java, TestNG, and Cucumber BDD following Page Object Model (POM) design pattern.

🛠 Tech Stack

Java

Selenium WebDriver

TestNG

Cucumber BDD

Maven

Git

JIRA

📂 Framework Architecture

Hybrid Framework combining:

Page Object Model (POM)

TestNG for execution

Cucumber for BDD

Data-driven testing

Maven for build management

📌 Modules Automated

Admin Login (Valid & Invalid scenarios)

Employee Registration (PIM Module)

User Registration

User Login

Logout functionality

📊 Test Coverage

Functional Testing

Regression Testing

Positive & Negative Scenarios

Scenario Outline with multiple test data

▶ How to Run

Clone the repository

Open in Eclipse / IntelliJ

Run TestRunner class

Execute testng.xml
